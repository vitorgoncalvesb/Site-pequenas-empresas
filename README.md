# Pedro Lima Coffee — Site Front-End

Site front-end premium e responsivo para a marca de café artesanal *Pedro Lima Coffee*.

## Estrutura

- `index.html` — Página inicial
- `shop.html` — Loja com filtros e carrinho
- `product.html` — Página de detalhes do produto
- `courses.html` — Cursos e workshops de café
- `subscription.html` — Planos de assinatura e preferências
- `blog.html` — Jornal / hub de educação
- `about.html` — História da marca e valores
- `contact.html` — Formulário de contato e links sociais
- `assets/css/style.css` — Design visual e layout responsivo
- `assets/js/app.js` — Interações, filtros, carrinho, alternância de tema

## Visualizar

Abra qualquer arquivo HTML no navegador para ver o site.

## Observações

- O site foi construído com HTML, CSS e JavaScript puro.
- Utiliza Google Fonts remotos e URLs de imagem externas para visuais elegantes.
- A página da loja tem um sistema de filtros interativo e painel de carrinho.
- A página de assinatura inclui alternância de preços e painéis de preferência.
